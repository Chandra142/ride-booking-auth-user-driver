# 🚖 Ride Booking - Auth, User & Driver Services

A Spring Boot Microservices project implementing the **Authentication**, **User Management**, and **Driver Management** modules of a Ride Booking System.

---

## 🚀 Tech Stack

- Java 21
- Spring Boot 3
- Spring Security
- JWT Authentication
- Spring Data JPA
- Hibernate
- MySQL
- Maven
- REST APIs
